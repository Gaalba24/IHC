# StudyBalance AI — Landing Page


## Descripción del proyecto
 
StudyBalance AI es una aplicación web diseñada para ayudar a estudiantes universitarios a gestionar su carga académica reduciendo el estrés mediante inteligencia artificial. La plataforma detecta el nivel de agotamiento del usuario, reorganiza tareas automáticamente y sugiere pausas activas para mantener un rendimiento sostenible.
 
Este repositorio contiene el **Landing Page** del proyecto, desarrollado como parte del curso de Interacción Humano-Computadora y Tecnologías Móviles (IHC) de la Universidad Peruana de Ciencias Aplicadas (UPC).
